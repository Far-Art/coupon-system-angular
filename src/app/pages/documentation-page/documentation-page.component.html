Features

  • Mobile First Approach
    This website built to excellently be displayed on mobile devices
  • Auto Dark Mode
    The user can switch between dark, light and device themes
  • Preferences Storage
    The user preferences stored locally on device, and for logged users also on server
  • Lazy Loaded Modules
    Modules are being loaded only when they are needed
  • Route Guards
    User cannot access routes he has no authority to and will be redirected to proper pages
  • Alert On Unsaved Data
    When user modifies account and tries to leave without saving, the prompt will appear
